**Learning Javascript**
0. Basics of JavaScript
-It is a case-sensitive language.

1. Variables --> name of the storage location
## syntax for declaring variables (let,const,var)
-let ---> re-assigned | Block scoped |cannot be re-declared
-const--> cannot be re-assigned and re-declared |Block scoped
-var----> can be re-assigned and re-declared | Block scoped

2. Data Types
-Number | String | Boolean | Null | Undefined | Symbol | Big int --> Primitive data types
-use "typeof" --> to check data type of variables
-Derived data types(Array, Object, function)

3. Operators 
-Arithmetic (+,-,*,/,%,**)
-%(modulus)--> takes out remainder
-**---> power
-Comparision/Relational (<,>,>=,<=,==,===,!=,!==)
-(==) checks only value
-(===) checks both value and type
-Assignment (=,+=,-=,*=,/=)
--> +=,-=,*=,/= (first add,sub ..... and assign)
-Logical (&& , || , !)
--> && (false if one of the condt is false)
--> || (true if one of the condt is true)
-Unary (++,--)
